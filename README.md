# Astro Boilerplate

## Commands

All commands are run from the root of the project, from a terminal:

| Command                | Action                                           |
|:-----------------------|:-------------------------------------------------|
| `yarn install`         | Installs dependencies                            |
| `yarn dev`             | Starts dev server in development mode            |
| `yarn dev:staging`     | Starts dev server in staging mode                |
| `yarn dev:prod`        | Starts dev server in production mode             |
| `yarn build`           | Build for production to `./build/`               |
| `yarn build:staging`   | Build for staging to `./build/`                  |
| `yarn build:dev`       | Build for development to `./build/`              |
| `yarn preview`         | Preview production build locally                 |
| `yarn preview:staging` | Preview staging build locally                    |
| `yarn preview:dev`     | Preview development build locally                |
| `yarn astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `yarn astro -- --help` | Get help using the Astro CLI                     |
